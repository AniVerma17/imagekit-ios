##Installation

CocoaPods

You can use CocoaPods to install ImageKit by adding it to your Podfile:

```ruby
use_frameworks!
platform :ios, '8.0'

target 'target_name' do
pod 'imagekit'
end
```

To use the ImageKit library in your app, import ImageKit into the file(s) where you want to use it.

```ruby
import imagekit
```
